# Minion-Hub
Modern laptops are becoming increasingly slimmer and lighter, often at the expense of having fewer built-in USB ports. Therefore, a USB hub is essential for expanding connectivity and accomodating multiple USB devices simultaneously. However, there is one common problem to implement this device, that is EXPENSIVE. 
A standard industrialised USB hub costs around 20 USD onwards, but what if I tell you that you could build one for just less than 10 USD.

# How it works
- Plug in a USB type C cable to the bottom USB type C upstream port to power up the USB hub.
- Use any 4 USB ports on the side of the USB hub to power up your devices simultaneously.
